В каталоге находятся 2 папки:

 1) NexArt - функционирующий сайт, который разрабатывается в настоящее время. Адаптирован под мобильные устройства.
             Доступен по ссылке http://nexart.pro

 2) Memoraps - Сайт-хранилище. Здесь я реализовал возможность создания папок для отдельных пользователей.
    В каждой папке, можно хранить различные файлы, загруженные с ПК или сохраненные по ссылке. Сайт не доделан, 
    но довольно много функционала встроено. Если смотреть с точки зрения сложности JavaScript, то этот проект интереснее чем первый.
    
    Разработка над проект преостановлена, т.к не вижу смысла делать сайт на которые не зайдет ни один человек.(т.к нет контента).
    Ввиду этой проблемы было принято решение перейти к созданию сайта NexArt.
